# Freestar 4000
FreeStar 4000 is a [Python](http://python.org/)-based high-accuracy [WeatherStar 4000](https://www.twcarchive.com/wiki/Weather_Star_4000) simulator.
Freestar 4000 Canadian edition is an attempt to recreate the Weatherstar 4000s used by The Weather Network.
